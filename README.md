Personal landing page for Oscar Paredes - Software Engineer & Technical Leader.

## Overview

A clean, Medium-inspired single-page portfolio showcasing professional experience, skills, and contact information.

## Tech Stack

- HTML
- CSS (no frameworks)

## Run Locally

Open `index.html` in a browser, or serve with:

```bash
npx http-server
```

Then visit `http://localhost:8080`
